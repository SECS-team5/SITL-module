# Архитектура системы управления дронами

## Текущий статус

Так как на наших плечах реализация SITL модуля с интеграцией ardupilot и Gazebo, было решено разбить на три этапа 

1) SITL "Light/Простой" 

Расчет базовых перемещений дронов

2) SITL Ardupilot

Автопилот для логистических дронов 

3) SITL PX4

Автопилот для аграрных дронов-опрыскивателей

4) Интеграция с Gazebo

Визуализация перемещений дронов

## Общая схема взаимодействия компонентов

Диаграмма показывает предполагаемые потоки данных между основными компонентами системы на текущем этапе разработки (разработка SITL "Light/Простой", реализованы заглушки интерфейсов).
